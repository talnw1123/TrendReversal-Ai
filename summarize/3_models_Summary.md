# เจาะลึกการทำงาน: `model/models.py`
**(พิมพ์เขียวสถาปัตยกรรมโครงข่ายสมองกล - Deep Learning Blueprint / Cell Library)**

ไฟล์นี้ทำหน้าที่เป็น **คลังอาวุธ หรือ โรงเก็บแบบแปลน (Blueprint)** สำหรับกระบวนการเรียนรู้เชิงลึก (Deep Learning) ที่จะถูกเรียกไปใช้งานต่อ
จุดสำคัญคือ ไฟล์นี้ **"ไม่ได้ทำการเทรนโมเดลเลยซักนิดเดียว"** แต่ทำหน้าที่ประกาศสร้างมิติ (Dimensions), เลเยอร์ (Layers), และวงจรประสาทเทียมระดับสูงที่ประกอบร่างกันเอาไว้ เพื่อให้สคริปต์หลัก (`train_separate_models.py`) มาดึงสถาปัตยกรรมเหล่านี้ไปใช้กับข้อมูลของตัวเอง

ในคลังแห่งนี้บรรจุ 4 สถาปัตยกรรมระดับแนวหน้าของโลก AI เอาไว้:

## 1. `build_lstm` (Bidirectional LSTM พร้อม Self-Attention)
เป็นสุดยอดโมเดลสำหรับการตีความ **"ข้อมูลลำดับและเวลา (Time Series / Sequence Data)"**
- **หลักการทำงาน:** ราคาสินทรัพย์ไม่ได้เกิดแบบสุ่มรายวัน แต่วันนี้มีส่วนเกี่ยวข้องกับราคา 60 วันที่แล้ว LSTM มีคุณสมบัติเรื่อง "Memory Cell" ทำให้มันจำแพทเทิร์นข้ามเวลาได้ยาว
- **สิ่งที่พัฒนาเพิ่ม:** 
  - ใช้แบบป้อนสองทิศทาง (Bidirectional) ทำให้ AI กวาดตามองความหลังและมองย้อนกลับไปกลับมา
  - เสริมทัพด้วย `MultiHeadAttention` (เหมือนหัวใจของ ChatGPT) ปลั๊กเข้าไป เพื่อให้มันบอกได้ว่าใน 60 วันย้อนหลัง มีแท่งเทียนแท่งไหนที่เป็นเหตุการณ์ "กุญแจสำคัญ (Key Catalyst)" ที่ต้องโฟกัส
  - มีระบบตัดเสียงรบกวนและป้องกันการท่องจำข้อสอบด้วย `Dropout(0.3)`

## 2. `build_cnn` (1D Convolutional Neural Networks แบบมีทางลัด)
เดิมที CNN คือราชันแห่งงานวิเคราะห์รูปภาพ แต่เมื่อบีบให้มาทำงานแนวยาวแบบกราฟราคา (1D) มันจะกลายเป็นสุดยอดนักจับ **"ลวดลายเฉพาะจุด (Local Patterns / Spikes)"**
- **หลักการทำงาน:** ทำตัวเหมือนฟิลเตอร์เอาไฟฉายสไลด์สท้อนไปตามกราฟ กวาดหาจังหวะที่ราคาหักศอก, จังหวะวอลุ่มพุ่งปรี้ด (Spike)
- **สิ่งที่พัฒนาเพิ่ม:** 
  - มีการต่อแบบสายฟ้าแลบ หรือ Residual Connection `Add()([x, residual])` เพื่อให้ข้อมูลตรรกะระดับลึก ไม่หล่นหายระหว่างลงไปยังเซลล์ประสาทชั้นล่าง ทำให้กราฟพยากรณ์มีความคมขึ้น ไม่ทื่อ

## 3. `build_transformer` (Transformer Encoder Blocks ขั้นสูง)
เทคโนโลยีสดใหม่ที่ปฏิวัติโลก AI ปัจจุบันถูกนำมาประยุกต์ใช้เพื่อดูภาพรวมกราฟยาวๆ ในแว็บเดียว
- **หลักการทำงาน:** แทนที่จะไล่อ่านกราฟ 1, 2, 3.. วัน เหมือน LSTM ทรานเมอร์ใช้วิธีมองกระดาน 60 วันทั้งหมด "ในเสี้ยววินาทีเดียวกัน" ผ่าน `transformer_encoder_block` 
- **สิ่งที่พัฒนาเพิ่ม:** 
  - อัดแน่นด้วย Multi-Head Self-Attention หลายชั้น ทำงานเหมือนมีนักวิเคราะห์ 4 คน (`num_heads=4`) คอยเพ่งกันคนละมุม (มุมวอลลุ่ม, มุมราคา, มุมระยะห่างฐาน, ฯลฯ)
  - ให้ผลลัพธ์ที่ดีเลิศหากโมเดลสามารถผสานความสัมพันธ์ที่ไกลเป็นเดือนเข้ากับความน่าจะเป็นวันพรุ่งนี้ได้

## 4. `build_mlp` (Enhanced Multi-Layer Perceptron)
แม้จะเป็นพื้นฐานสุด แต่ MLP คือคนงานที่ถึกทนและเชื่อถือได้ตลอดกาล
- **หลักการทำงาน:** รับตัวเลขหน้ากระดานปัจจุบันมาตีกระจาย (Flatten) และส่งเข้าสมองส่วนลึกหลายชั้น (`Dense Layer -> 256 -> 128 -> 64`)
- เหมาะอย่างยิ่งถ้าระบบทำการคัดกรองตัวแปรออกมาได้เด็ดขาดแล้ว (สกัดด้วย feature_importance แล้ว) ก้อนสมองอัดหนาๆ แบบนี้จะสามารถเชื่อมโยงหลักการแบบ Non-Linear หาจุดระเบิดทำกำไรออกมาได้อย่างรวดเร็ว

### การตั้งค่ามาตรฐาน (Output)
ทุกพิมพ์เขียว จะถูกตั้งค่าให้ออกคำตอบสุดท้ายด้วย `Dense(num_classes, activation="softmax")` ซึ่งก็คือการหักดิบตีค่าออกมาเป็นเปอร์เซ๊นต์ความน่าจะเป็นในกรอบการตัดสินใจ:
- **Binary Mode (ดวลสองหน้า):** บอกเปอร์เซ็นต์ { 🟢 ซื้อ(ขึ้น), 🔴 ขาย(ลง) }
- **Multi-class Mode:** บอกเปอร์เซ็นต์ { 🟢 ซื้อ, 🔴 ขาย, ⚪ รอดูใจเฉยๆ }
